# Sensor-corruption

## Creating a python environment
First create a virtual python environment
```
$ python3 -m venv .venv
```

Activate environment
```
$ source .venv/bin/activate
```

Install required libraries
```
(venv) $ pip install numpy ...
```

Run files
```
(venv) $ python src/data.py
```

Deactivate
```
(venv) $ deactivate
```
